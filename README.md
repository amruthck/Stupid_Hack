# 🚀 DeserveMaps — Earn Your Route

> A brutally honest navigation system where your intelligence decides your path.

---

## 🧠 Concept

**DeserveMaps** flips traditional navigation.

Instead of giving the fastest route blindly, it **tests the user first**:

- Smart answers → 🚀 Shortcut  
- Average answers → 🟡 Exploration route  
- Wrong answers → 💀 Punishment route  

Navigation becomes a **game + judgment system**.

---

## 🎯 Problem Reimagined

Current maps:
- Treat all users equally  
- Optimize only for speed  

**DeserveMaps:**
- Introduces behavior-based routing  
- Makes navigation interactive and fun  
- Turns a boring task into an experience  

---

## ⚙️ How It Works

1. User selects:
   - Destination (Library, Canteen, etc.)
   - Transport mode (Walk / Bike / Car)

2. System runs an **Absurdity Test**:
   - 3 questions  
   - Scoring:
     - Correct → +20  
     - Wrong → -10  

3. Route Assignment:
   - Score > 60 → 🟢 Shortcut  
   - 30–60 → 🟡 Normal  
   - < 30 → 🔴 Long route  

4. Bonus Twist:
   - Midway in long route → redemption question  

5. Live Simulation:
   - Animated route  
   - Moving marker  
   - Progress tracking  
   - Dynamic directions
